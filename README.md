# CRUD de livros com NodeJs

## Projeto com banco de dados MySQL para realização de CRUD com livros.

---

### 📋 Pré-requisitos

É necessário instalar o 'express', 'handlebars' e o 'mysql'

```
`npm i express, handlebars, mysql`
```

### ⚙️ Para iniciar o Programa

```
`npm start`
```
